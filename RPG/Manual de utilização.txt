##### As aventuras de Haddad, Putin e Lula #####


	Requisitos:
	> Qualquer Sistema Operacional (SO)
	> Java Runtime Enviroment (JRE)
	> É possível que seja necessário possuir o Java Developer Kit (JDK)
	> Você pode: 
		importar o projeto "RPG" na IDE de sua preferência e executá-lo
		no terminal ou similar, inserir o comando: java -jar "[CAMINHO DA PASTA]\RPG\dist\RPG.jar"

	COMO JOGAR
	> O jogo funciona a partir de interações via texto com o jogador
	> Possui um conceito de mapa e navegabilidade, onde o jogador insere uma opção para efetuar uma ação
	> O jogador escolhe um dos três personagens disponíveis (a história não se altera).
	> Os personagens possuem atributos distintos
	> O objetivo do jogo é encontrar o vilão Biroliro e ser vitorioso no embate contra ele
	> Ações durante a batalha ocorrem a partir do rolar de um dado.

Bom jogo!
